# AWS Cli

AWS cli és una API que ens permet comunicar-nos amb la infraestructura de Amazon Web Services. A través de l'API podem **crear i gestionar** les nostres màquines d'AWS.

# Instal·lació

Necessitem una sèrie d'eines per a poder comunicar-nos amb comandes amb la infraestructura d'AWS. En la següent plana web s'explica quines eines necessitem i com instal·lar-les.

https://docs.aws.amazon.com/cli/latest/userguide/getting-started-install.html

# Operacions

Abans de poder utilitzar l'API d'AWS en el terminal, hem de fer una sèrie d'operacions. Cal que fem aquestes operacions cada vegada que arrenquem el laboratori d'AWS.

# Prova de funcionament

Executa en el teu terminal les comandes:

```bash
aws configure get region
aws ec2 describe-security-groups
```
que t'haurien de retornar la regió que tens assignada i els grups de seguretat definits en el sistema.
