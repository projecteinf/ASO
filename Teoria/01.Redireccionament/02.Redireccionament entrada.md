# Redireccionament entrada

- Utilitzem "<"

# Exemples utilització de redireccionament

```bash

    cat < fitxer.txt    # equivalent a cat fitxer.txt

    sort < fitxer.txt    # equivalent a sort fitxer.txt

    tr a-z A-Z < fitxer.txt # Passa totes les lletres minúscules a majúscules (alfabet anglès a-z)

```

- La principal diferència és que tr no accepta un fitxer com a argument.

    - tr llegeix de l'entrada estàndard.

# Consideracions

    - Els nostres programes MAI demanaran dades a l'usuari.  

    - Utilitzem < quan volem enviar dades al nostre programa.  

---

# Exemple utilització

Fitxer: llegir.sh

```bash
#!/bin/bash

echo "Comença l'script!"
echo "Reb línies per stdin (entrada estàndard)."

# Llegeix línia a línia de l'entrada
while read linia
do
    echo "He llegit: $linia"
done

echo "Fi de l'script!"
```

---

# Fitxer d’entrada: `entrada.txt`

```
Hola
Això és una prova
Línia número 3
```

---

# Execució amb redirecció `<`

```bash

bash llegir.sh < entrada.txt

```
