# Redirecció de sortida

* Per defecte -> sortida estàndard (`stdout`).

* Utilitzem > i >> per guardar la informació en un fitxer en lloc de mostrar-la per la pantalla.  

---

# `>` (sobreescriure)

* Crea un fitxer nou amb la sortida.  

* Si el fitxer ja existeix, el sobreescriu (esborra el contingut anterior).  

```bash

    echo "Hola" > salut.txt

```

# `>>` (afegir)

* Escriu al final del fitxer.  

* Si el fitxer no existeix, el crea.  

* Si existeix, afegeix el text sense esborrar el que hi havia.  


```bash

    echo "Bon dia" >> salut.txt

```

---

# Exemple comparatiu

```bash

ls > llista.txt      # Sobreescriu llista.txt amb el resultat de ls

ls >> llista.txt     # Afegeix el resultat de ls al final del fitxer

```
