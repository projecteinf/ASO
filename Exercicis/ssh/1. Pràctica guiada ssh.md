## Part 4 — Clau pública

4. Genera una clau SSH:

```bash
ssh-keygen
```

5. Copia-la al servidor:

```bash
ssh-copy-id isard@172.20.1.1
```

6. Torna a connectar:

```bash
ssh isard@172.20.1.1
```

### Preguntes

* Has introduït password?  

* On es guarda la clau pública al servidor?  

* Quin avantatge té aquest sistema?
